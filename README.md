# Archive Templates

This repository is an archive of scratch files, design explorations, and one-off
templates I have gone through while experimenting with personal sites and study
trackers.

It is kept for reference only. The files here are not intended to be maintained
as production code, reused as active project scaffolding, or treated as the
source of truth for any live site.

## Contents

- HTML prototypes and direct-rendered experiments
- JSX design labs and layout explorations
- Static assets used by those prototypes
- Archived tracker or page concepts that have since moved elsewhere

## Notes

- Expect rough edges, duplicate variants, and unfinished ideas.
- Prefer active project repositories for current source files.
